# Neon Salvage

Neon Salvage is a small browser arcade game created for the Lecture 3 open-ended AI-assisted app/game development task.

## How to Run

Open `index.html` in a web browser. No installation or internet connection is required.

## Game Idea

The player pilots a salvage ship inside a neon space station. The goal is to collect power cells, keep energy above zero, and avoid sentry drones. The game becomes harder as the player collects more cells.

## Features

- Canvas-based animated gameplay
- Custom SVG favicon
- Keyboard controls and pointer/touch movement
- Score, level, energy, and best-score tracking
- Increasing difficulty with extra drones
- Shield ability with cooldown
- Restart and pause controls
- Responsive layout for desktop and mobile

## Controls

- Move: `WASD` or `Arrow Keys`
- Shield: `Space` or double-click/tap the game area
- Pause: `P`
- Mobile: drag on the game area to steer
